The system reads the file using PyPDFLoader.Chunking: The text is split into smaller chunks (2000 characters) using RecursiveCharacterTextSplitter.Retrieval (TF-IDF): instead of converting text to heavy vectors, the system indexes the chunks using statistical keyword frequency.
